Настройка MacOS ElCapitan для разработки

Segei Manin Опубликовано 2023-01-06 включен в Configuration
1. Настройка клавиатуры Motospeed Ck104 Заходим в System Preference/Keyboard/Modifier Keys, там выбираем USB Gaming Keyboard. И меняем клавиши Control и Option местами, чтобы было соотвествие расположения клавиш Apple Keyboard: 2. Установка XCode и Command Line Tools Максимальная версия XCode доступная для установки на MacOS ElCapitan 8.2.1. Скачиваем XCode и соотвествующую ему версию CLT c офиц.сайт Apple Устанавливаем XCode 8.2.1 из файла Xcode_8.2.1.xip и CLT из файла Command_Line_Tools_macOS_10.

Чистая установка MacOS ElCapitan

Segei Manin Опубликовано 2023-01-05 включен в Configuration
1. Скачиваем установщик MacOS ElCapitan Страница сo всеми доступными версиями MacOS на офиц.сайте Ссылка загрузки MacOS ElCapitan 10.11.6 2. Установка установщика MacOS ElCapitan Дважды щелкаем на скаченный файл .dmg. Дважды щелкаем на файл .pkg. Следуем инструкциям. Установщик MacOS ElCapitan будет установлен в папку Applications. 3. Создание загружаемого установщика на флеш-накопителе USB или встроенном томе Флеш-накопитель USB или другой том должен иметь емкомсть не менее 14 Гб и быть отформатированным с файловой системой MacOS Extended (Journaled).
