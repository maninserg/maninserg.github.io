1. Настройка клавиатуры Motospeed Ck104

Заходим в System Preference/Keyboard/Modifier Keys, там выбираем USB Gaming Keyboard. И меняем клавиши Control и Option местами, чтобы было соотвествие расположения клавиш Apple Keyboard:

2. Установка XCode и Command Line Tools

Максимальная версия XCode доступная для установки на MacOS ElCapitan 8.2.1.

Скачиваем XCode и соотвествующую ему версию CLT c офиц.сайт Apple

Устанавливаем XCode 8.2.1 из файла Xcode_8.2.1.xip и CLT из файла Command_Line_Tools_macOS_10.11_for_Xcode_8.2.dmg

3. Установка MacPorts и пакетов

Страница со всеми доступными версиями MacPorts

Ссылка на скачивание MacPorts для MacOS ElCapitan

Запускаем файл .pkg и следуем инструкциям по установке.

Страница с доступными Ports

Устанавливаем следующие пакеты:

  • Инструменты для разработки:
    • sudo port install git details
    • sudo port install vim details
    • sudo port install docker details
  • Языки программирования:
    • sudo port install python310 details
    • sudo port install py310-setuptools details
    • sudo port install nodejs18 details
    • sudo port install php82 details
    • sudo port install ruby31 details
    • sudo port install go details
  • Базы данных:
    • sudo port install sqlite3 details
    • sudo port install mysql57 details
    • sudo port install postgresql15 details
    • sudo port install mariadb details
    • sudo port install redis details
  • Web-серверы:
    • sudo port install apache2 details
    • sudo port install nginx details
  • Среды разработки:
    • sudo port install Arduino details
  • Инструменты для работы с сетью:
    • sudo port install curl details
    • sudo port install nmap details
    • sudo port install wireshark3 details
    • sudo port install elinks details
  • Инструменты для работы с видео:
    • sudo port install youtube-dl details
    • sudo port install ffmpeg details
    • sudo port install VLC details
  • Системные утилиты:
    • sudo port install htop details
    • sudo port install neofetch details
    • sudo port install nnn details
  • Инструменты для работы с графикой:
    • sudo port install gimp2 details
    • sudo port install ImageMagick details

4. Установка браузеров Chrome и Firefox

Доступная версия Google Chrome для MacOS ElCapitan 102.0.5005.63. Скачиваем с офиц.сайта Google. И устанавливаем из скачанного файла google-chrome-102-0-5005-63.pkg.

Доступная версия Firefox для MacOS ElCapitan 78.15.0esr. Скачиваем с офиц.сайта Firefox - русская версия, офиц.сайта Firefox - английская версия. Устанавливаем из скачанного образа Firefox 78.15.0esr_ru.dmg или Firefox 78.15.0esr_en_US.dmg

Устанавливаем расширения для Firefox РуТрекер - официальный плагин (доступ и пр.), AdBlocker Ultimate, Print Edit WE, SingleFile, TWP - Translate Web Pages

Доступная версия Visual Studio Code для MacOS ElCapitan 1.74.2, скачиваем ее с офиц.сайта.

Разархивируем zip-архив и копируем приложение в Applications
